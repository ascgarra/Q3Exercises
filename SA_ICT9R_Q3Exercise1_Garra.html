HTML

HyperText Markup Language

It's a language used to create simple, static websites. HTML is a set of tags that indicate the structure of a document. HTML documents are viewable using web browsers, but they can also be dynamic and interactive with modern technology.

CSS

Cascading Style Sheets

This is a style sheet language that describes the way a document should look if it's written in markup. It's a file with rules that define how text and other elements should be formatted. CSS is used for formatting web pages, but it can also be used to format presentations and printable documents. It's commonly used to format web pages, but it can also be used to format other types of documents, like presentations.

Bootstrap

Bootstrap Version 4

It is a responsive front-end framework developed by Twitter and released as open source. Bootstrap 4's main feature is its modularity and flexibility. Due to its open-source nature, Bootstrap is constantly updated with new features by an active developer community. Your project can also be customized and extended with Bootstrap's framework.
